# 🏙️ MZ 유입 잠재력이 가장 높은 서울시 행정동 예측

## 📋 프로젝트 개요

### 가설

> **"검색 트렌드와 상권 특성 데이터를 결합하면 MZ 유입 잠재력이 큰 상권을 조기에 발굴할 수 있다"**

### 목표

- MZ세대 유입 잠재력이 가장 높은 **행정동 예측**
- 기존 핫플레이스(성수, 을지로 등)의 **공통 특성 추출**
- 데이터 기반 **Post 핫플레이스** 선정

### 분석 과제

| 과제       | 설명                 | 결과물                 |
| ---------- | -------------------- | ---------------------- |
| **과제 1** | 하이브리드 모델 개발 | Post 핫플레이스 TOP 50 |
| **과제 2** | 사분면 분석          | 행정동 4개 그룹 분류   |
| **과제 3** | 통계적 검증          | 카이제곱 검정          |

---

## 🔬 분석 방법

### 1. 하이브리드 모델

서울 전체 데이터 + 핫플레이스 특성을 결합한 점수 모델

```
하이브리드 점수 = Model2(70%) + 핫플 유사도(30%)
```

→ [상세 문서](docs/hybrid_model.md)

### 2. 사분면 분석

가중합산 스코어링으로 행정동을 4개 그룹으로 분류

```
X-Index = CAGR(34%) + 네이버검색(33%) + 블로그(33%)
```

→ [상세 문서](docs/quadrant_analysis.md)
