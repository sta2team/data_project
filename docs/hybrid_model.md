# 🔬 하이브리드 모델 상세

MZ세대 유동인구 증가율을 예측하여 **Post 핫플레이스**를 선정하는 모델입니다.

---

## 모델 구조

```
하이브리드 점수 = Model2_점수_정규화 × 0.7 + 핫플_유사도_점수_정규화 × 0.3
```

| 모델 | 데이터 | Prob(F) | 역할 | 가중치 |
|------|--------|---------|------|--------|
| Model 2 | 서울 전체 (11,443개) | 0.0001 ✅ | 통계적 신뢰성 확보 | **70%** |
| 핫플레이스 모델 | 성수/을지로/송파/문래 (216개) | 0.0281 ✅ | 핫플 특성 반영 | **30%** |

### 가중치 설정 근거

- **Model 2**: 서울 전체 데이터로 학습, 통계적으로 유의 (P < 0.05)
- **핫플레이스 모델**: 검증된 핫플레이스 4개동의 공통 특성 추출

---

## 정규화

두 모델의 점수 스케일이 다르므로 **MinMaxScaler**로 0~1 정규화 후 결합

| 점수 | 정규화 전 범위 | 정규화 후 범위 |
|------|---------------|---------------|
| Model2_점수 | -0.0197 ~ 0.0149 | **0 ~ 1** |
| 핫플_유사도_점수 | -0.0621 ~ 145.99 | **0 ~ 1** |
| 하이브리드_점수 | - | **0 ~ 0.70** |

### 왜 기존 핫플이 상위권이 아닌가?

**천장 효과 (Ceiling Effect)**

- 이미 MZ 유동인구가 포화 상태
- **증가율**이 제한됨
- 모델은 "현재 핫한 곳"이 아닌 **앞으로 성장할 곳**을 예측
